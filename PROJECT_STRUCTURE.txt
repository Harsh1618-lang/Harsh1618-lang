Quick Start Steps
-----------------
1. flutter pub get
2. flutterfire configure
3. firebase deploy --only firestore:rules,storage
4. Add Agora App ID in lib/core/live_service.dart
5. flutter run
6. Create admin user (change role in Firestore)
